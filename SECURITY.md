# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Blackdoor Industries repository, please report it responsibly.

**Email:** security@blackdoor.com

Alternatively, open a [private security advisory](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) on the affected repository via GitHub.

Please do not open public issues for security vulnerabilities.

## Response

We will acknowledge receipt within 48 hours and provide a timeline for resolution.
